Principais formatações

Negrito / Destaque

Nessa frase temos um Termo em negrito usando a tag
<b> Termo em negrito (esta tag não é semântica) </b>

Nesta frase temos um Termo em destaque usando a tag
<strong>Termo em destaque (esta tag é semântica)</strong>

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag
<i> termo em itálico (esta tag não é semântica) </i>

Nesta frase temos um termo em itálico usando a tag
<em> termo em itálico (esta tag é semântica) <em>

Atalho do vscode

Podemos utilizar o comando ctrl + shift + p do vscode e utilizar a função wrap with abbreviation

Texto marcado

Podemos criar também um texto marcado usando a tag
<mark> um texto marcado </mark>

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Criamos estes textos utilizando as tags
<big> texto grande </big>
<small> texto pequeno </small>

Vale lembrar que a tag <big> foi descontinuada por ser considerada apenas como forma sem um significado, podendo ser aumentado o seu tamanho utilizando CSS.

Texto deletado

Podemos marcar um texto como excluído para indicar como ele deve ser lido, mas não considerado.

Este texto marcado como excluído foi usada a tag
<del> um texto excluído </del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Este texto marcado como inserido foi usada a tag
<ins> um texto como inserido </ins>

Texto sublinhado

Existe também o sublinhado com a tag U (não semântica)

Este texto marcado como sublinhado foi usada a tag
<u> sublinhado </u>

Texto sobrescrito

Para inserir coisas do tipo x20+3

Este texto sobrescrito foi utilizada a tag
x<sup>20</sup>+3

Texto subscrito

Para inserir coisas do tipo H2O

Este texto subscrito foi utilizada a tag
H<sub>2</sub>
