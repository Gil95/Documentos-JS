O que é HTML, CSS e JavaScript?

HTML

Agrega todo tipo de Conteúdo
Textos, imagens, vídeos e tabelas.

CSS

Agrega Desing ao conteúdo
Cores, sombras, tamanhos e posicionamento.

Javascript

Agrega Interações ao conteúdo
Menus, animações, popups e validações.
