Operadores de matemática

Operadores aritméticos básicos.
+ = soma
- = subtração
* = multiplicação
/ = divisão real ( com valores após a vírgula )
% = resto da divisão ( 5/2=2, resta 1 que é o resultado )
** = potência ( 5**2 = 25 )

Incremento

Podemos somar +1 ou subtrair -1 com um comando simples e rápido
"n = n + 1" é o mesmo que "n += 1", para somar +1 podemos usar n++ ou ++n
o mesmo acontece usando n-- ou --n
a diferença entre n++ ou ++n é que, o comando ++n mostra o incremento instantaneamente
