Sinais de operadores relacionais

> = maior que
< = menor que
>= = maior ou igual
<= = menor ou igual
== = igualdade
!= = diferente Podemos fazer operações aritméticas antes de relacionar as variáveis

Operadores de identidade

5 == '5' (verdadeiro) dessa forma, ele testa somente o valor
5 === '5' (falso) dessa forma, ele testa também o seu tipo
5 != '5' (falso) testando somente o valor, ele diz que não são diferentes
5 !== '5' (verdadeiro) testa também o tipo, informando que são diferentes

Operadores lógicos

! Negação (não quero isso)
&& Conjunção (quero todos esses)
|| Disjunção (pode ser qualquer um desses)

A ordem dos operadores lógicos é sempre de "Negação", "Conjunção" e então "Disjunção"
Os operadores lógicos somente respondem operadores relacionais

Operador ternário

?: atribui um valor true ou false dependendo do resultado do designado no início da operação

[TrueOrFalse]?[True]:[False]
O operador ternário é o ultimo na ordem dos operadores.
