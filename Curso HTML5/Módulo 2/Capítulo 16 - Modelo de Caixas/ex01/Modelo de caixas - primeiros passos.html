Como funciona um modelo de caixas

Utilizando o modelo de caixas

Utilizamos algumas propriedades para dar algum valor às tags em HTML, e para criar caixas não seria diferente.
Podemos utilizar propriedades como:

background-color
Define a cor de fundo de uma tag HTML
margin
Define o espaço das laterais, sendo apenas margin, para todos os lados, caso deixe em automático, ele centraliza a caixa e define um espaço para os lados
border-radius
Define uma borda arredondada para o modelo de caixa
padding
Define um preenchimento para o modelo
box-shadow
Define uma sombra para o modelo definindo a localização e de desvanecimento da sombra, além de definir a sua cor.
height
Define a altura do modelo
width
Define a largura do modelo
color
Define a cor do conteúdo exibido dentro do modelo
outline
Define o cortorno do modelo

Tipos de caixas

Existem 2 tipos de caixas:

box-level
Uma caixa que define seu conteúdo em parágrafos, ocupando toda a largura do site.
inline-level
Uma caixa que mostra seu conteúdo em meio a linha do parágrafo, onde continua o conteúdo logo em seguida sem pular o parágrafo.
