Seletores id e class

O que são os seletores?

Podemos utilizar seletores, personalizando algumas tags de nosso HTML. Estes seletores são chamados de id e class, estas tags servem para identificar e personalizar algumas das nossas tags em HTML, como por exemplo, demarcar uma ou mais dentre outras tags de mesmo nível para serem personalizadas.

Um dos exemplos é as caixas main abaixo, com background-color de cores diferentes.

Como representar id e class em HTML

Em HTML devemos identificar os seletores dentro da tag de abertura, exemplo: 


<h1 id="principal"> Título de exemplo </h1>

Em CSS devemos representar os seletores id e class da seguinte forma 

h1#principal{
    text-align: center;
}

  • class deve ser representado como .
  • id deve ser representado como #

Qual devo usar? id ou class?

Use os seletores de forma correta

Quando usamos id, não deve ser repetido o mesmo id para diversas tags.
Neste caso, usamos o seletor class.

Seletor id
Deve ser usado de forma única
Seletor class
Pode ser usado em múltiplos elementos

Nomeie seus seletores de forma correta

Sempre que nomear seus seletores, nomeie de forma em que os classifique, mas nunca por sua forma indicada em seu CSS.

Podemos também indicar duas classes para uma mesma tag, basta colocar dentro do seletor class a segunda classe desejada, sem vírgula, apenas um espaço.

Podemos utilizar id e class na mesma tag, porém as CSS de id sobrepõem as CSS de class.
