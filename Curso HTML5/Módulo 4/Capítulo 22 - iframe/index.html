O que é um iframe?

O iframe é um quadro que possui uma display inline, ou seja, o iframe fica lado a lado com o texto

Configurando iframes

Por padrão, um iframe tem uma medida de 300px por 150px

Podemos a configurar da forma que quiser à partir das CSS, utilizando do seu seletor iframe, ou criando uma classe ou id para a mesma e utilizando de seu seletor .(classe) ou #(id).

Conteúdo local no iframe

O iframe pode ser endereçado por uma url interna de pastas, quanto de uma url externa de sites diferentes.

O iframe tem uma tag bem simples.

<iframe src= "../../Módulo 3/Capítulo 20 - Mini-projeto cordel/index.html" frameborder="0"> </iframe>


Esta tag acima se representa da forma a seguir.

Navegação no iframe

Podemos usar o iframe como uma tela para o nosso conteúdo, seguindo com um link e uma target definida da seguinte forma

<a href = "https://gil95.github.io/Meu-perfil" target="janela"> </a>

Esta "janela" deve ser o nome do iframe que servirá como navegador em nossa página.

<iframe src= "" frameborder="0" name="janela"> </iframe>

Dessa forma teremos algo do tipo.

  • Página 1
  • Página 2
  • Página 3


O srcdoc deve ser digitado da seguinte forma

<iframe srcdoc = "<h1>Título</h1> <p>Parágrafo</p>> </iframe>

A imagem também deve ser lembrada de como deve ser digitada, pois a mesma se usa apenas 'aspas simples'

<iframe srcdoc= "<img src='imagens/paisagem.jpg'> </iframe>

Tornando iframes mais seguros

O iframe possui um problema quanto a segurança, pois o mesmo pode fazer uma conexão para outros sites que não seja o seu. Este problema costuma estar relacionado à sites que não podemos controlar, visto que não é relacionado com os nossos, e a segurança do site de terceiros embora não seja nossa, a segurança de seus usuários é de nossa responsabilidade, pois o mesmo problema foi acessado à partir de nosso site.

Sandbox

A sandbox é uma propriedade do iframe que nos permite negar ou permitir alguns recursos de sites de terceiros, como script, formulários e etc. Caso a propriedade sandbox estiver com um valor sandbox, a mesma nega todo e qualquer tipo de permissão, sendo uma segurança de que até mesmo nos impede de utilizar suas interações por javascript ou interagir com seus links.


<iframe src = "paginas extras/pag004.html" target="janela" sandbox="sandbox" frameborder="1"> </iframe>

Política de referência

A política de referência remete à basicamente qual a relação temos com o site indicado, o mesmo deve ser indicado pela propriedade referrerpolicy com o valor de noreferrer

<iframe src = "paginas extras/pag004.html" target="janela" referrerpolicy="noreferrer" frameborder="1"> </iframe>

Desta forma, o conteúdo terá acesso à interações dos sites, mas terá uma política de não referenciado, ainda assim o site terá uma vulnerabilidade caso o usuário tenha permitido uma coleta de dados por parte do site de terceiro.
