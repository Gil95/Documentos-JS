O que são as Media Queries?

As Media Queries, de acordo com a MDN (Mozilla Developer Network), "consiste de um media type e pelo menos uma expressão que limita o escopo das folhas de estilo usando media features, tal como largura, altura e cor. Media queries, adicionadas no css3, deixam a apresentação do conteúdo adaptado a uma gama específica de dispositivos não precisando mudar o conteúdo em si."

No caso, o conteúdo se adapta à tela em que está sendo exibido, alterando sua forma em que é apresentado ao usuário, seus ícones, suas imagens e muito mais.

Preparando um site para a impressão

O site quando impresso não faz sentido que tenha links e menu de navegação, mas graças ao CSS por link, podemos usar uma página de CSS separada para a impressão.

Desta forma, formatamos as CSS's preparada a impressão, retirando da página impressa o menu de navegação, tornando páginas mais legíveis para a impressão.


O link das CSS's deve ser em dois padrões.
<link rel="stylesheet" href="estilos/style.css" media="screen">
<link rel="stylesheet" href="estilos/print.css" media="print">

Enquanto um link é designado pela propriedade media para a tela, outra CSS é designada para a impressão, tornando duas páginas de estilos diferentes.

O que são Media Features?

Utilizando um recurso de Media Features, podemos dar páginas de estilos diferentes de acordo com a sua necessidade.

Deixe-me adivinhar, sua tela está em modo paisagem.

Deixe-me adivinhar, sua tela está em modo retrato.

Na verdade sua tela apenas foi selecionada para mostrar o texto da forma que escolhi, pois a página de estilos selecionada é a referente ao formato utilizado, o texto na verdade foi escrito em duas formas diferentes como está sendo mostrado abaixo.

Media Features nos Links das CSS

As Media Features são basicamente características das media="types", Seguem de um tipo de media e complementada com uma característica como por exemplo:

  • Largura da tela

O que é Mobile First?

Mobile First é basicamente um conceito onde um site é criado pensando primeiro em dispositivos móveis.

Esse conceito foi criado por Luke Wroblewski em 2009, onde o mesmo defendia que um site tem de ser criado pensando primeiro em dispositivos móveis, e então seguir para os desktops, visto que os atuais dispositivos móveis são usados hoje em larga escala, mas à época, já era promissora a idéia de evolução dos smartphones que temos hoje em dia.

Como a proposta de um site mobile first é basicamente desenvolver já pensando em um smartphone, devemos começar com o estilo de um site mobile, ou seja, nosso CSS deve começar como padrão para um smartphone, e seguindo para os divertos tipos de dispositivos à partir de Media Queries.

O que são Device Breakpoints?

São marcações limites de telas, essas Device Breakpoints são os limites de telas, que vão desde Smartphones até as grandes telas de televisores.

Os device Breakpoints são definidor por media query, onde definimos um max-width e min-width para diversos tipos de estilos, quebrando o texto e tornando o mesmo mais responsivo, a tela se torna adaptável ao dispositivo e consequentemente, melhor visualização e interação.
