Formulários

O que é um formulário?

Um formulário é basicamente tudo aquilo onde você preenche com seus dados ou com alguma informação importante para que seja guardada, em nosso caso, para um servidor, seja uma plataforma ou website.

Método GET e POST

Método GET

O Método GET é uma forma de transmitir um formulário, porém o mesmo é enviado sem amostrado na URL do site, algo que não é nada seguro quando se trata de dados confidenciais, como documentos e senhas.

Método POST

O Método POST também é uma forma de transmitir um formulário, porém desta forma não é amostrada na URL do site, porém não se trata de segurança, apenas está de forma oculta da URL, mas ainda é possível encontrar o dado enviado.
A solução correta para a segurança desses dados, é a criptografia via HTTPS.

Quando usamos um ou outro?

GET
Quando desejamos que a URL seja compartilhável.
Quando os dados NÃO SÃO confidenciais
Possui um limite máximo de 3000 caracteres
POST
Quando possui fotos no formulário
Quando os dados SÃO confidenciais.
Quando desejamos que a URL não seja compartilhável
Não possui limite de caracteres

Elementos de formulários

Existes diversas propriedades de um formulário, todas devem ser digitadas dentro da tag <input>

Tags de formulários

Um formulário é composto de algumas tags principais:

form
A tag principal, que abrirá um formulário.
label
É a etiqueta para um input.
input
São caixas que irá interagir com o formulário, nessa tag teremos diversos tipos de caixas.

Tipos de input

Assim como temos diversas tags de formulários, temos diversos tipos de input.

text
Neste tipo de input, temos uma caixa para texto, que será usada para obter os dados para o formulário.
number
Neste tipo de input, temos uma caixa de número, que também será usada para obter os dados para o formulário.
submit
Neste tipo de input, temos uma caixa do tipo botão de envio, onde será efetuado o registro do formulário.
reset
Neste tipo de input, temos uma caixa do tipo botão de reset, para que sejam limpas as caixas, e então preenchidas novamente.
password
Neste tipo de input, temos uma caixa de texto, porém com os dígitos ocultos para uma senha não visível por terceiros ao redor do usuário.

Propriedades de formulários

Alguns exemplos de propriedades de formulários são:

name
identificação da input importante para o HTML
id
identificação da input importante para o JavaScript e Formulários, esta caixa é responsável para se identificar para uma label
required
Esta é uma propriedade de que torna o input obrigatório
min
Esta propriedade é utilizada em input do tipo number, onde o valor desta propriedade será o valor mínimo aceitado para este formulário.
max
Esta propriedade também é utilizada em input do tipo number, onde o valor desta propriedade será o valor máximo aceitado para este formulário.
minlenght
Esta propriedade é utilizada para definir a quantidade mínima de caracteres para este input.
maxlenght
Esta propriedade é utilizada para definir a quantidade máxima de caracteres para este input.
