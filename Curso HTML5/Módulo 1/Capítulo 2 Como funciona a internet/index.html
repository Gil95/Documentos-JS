Como funciona a internet?

Conexão entre o Cliente-DNS-Servidor

Os servidores são nomeados como um IP (Internet Protocol), que é basicamente um número identificador dado ao computador ou servidor ao conectar-se à rede.
o IP é definido pelo seu provedor de internet.

Para ser acessado de forma prática, faz-se o uso de servidores DNS que à partir de URLs, nos retorna o endereço de IP do servidor desejado.

Domínio (URL)

O domínio é único.
O domínio é pago anualmente.
Existem vários TLDs.

Existem diferentes tipos de TLDs (Top-Level Domain)
Assim como existem; GTLDs (Generic Top-Level Domain) e CCTLDs (Country Code Top-Level Domain)

Hospedagem

É o espaço para armazenar arquivos.
O serviço é pago mensalmente.
É escolhido de acordo com o espaço, memória e recursos.

O servidor possui um valor fixo de IP, que pode ser acessado à partir de uma URL cadastrada em um servidor DNS.
