As tags <pre> são feitas para respeitar as indentações do código digitado no editor de HTML.
Já as tags <code> são usadas para monoespaçar as letras de forma que seja melhor identificadas as letras escritas no HTML, ou seja, espaçar de forma uniforme as letras, respeitanto o mesmo tamanho em todas.
