Qual a diferença entre Front-end, Back-end e Full-stack?

Front-end


Desenvolvedores Front-end são responsáveis pela programação client-side, programação específica de HTML, CSS e JavaScript voltada ao cliente (usuário).

Os arquivos são enviados de um servidor e exibidos no computador do usuário.

Back-end


Desenvolvedores Back-end são responsáveis pela programação server-side, programação específica voltada ao servidor

Exemplos de linguagens server-side:
PHP, JavaScript (Node), C#, Python, Ruby e Java

Interação entre client-side e server-side

Enquanto o client-side ordena entre a organização, amostragem e estilos para o uso do cliente, tornando o uso do site mais agradável.
O server-side está ordenando banco de dados dos itens à ser amostrado do lado client-side.

Full-stack

Quando um desenvolvedor é especializado tanto em Front-end quanto em Back-end, este é chamado de Full-stack.
